# Retro Snake Game (JavaScript, HTML, CSS) | [Play Now](https://kolonatalie.github.io/snake/)
Classic Snake Game built with vanilla JavaScript, HTML &amp; CSS. Playable with keyboard or touch, fully responsive for desktop &amp; mobile. Features animated mobile menu, live score tracking, best score saving, and smooth gameplay.

## Features

- Cross-device support (desktop & mobile).
- Keyboard controls: arrow keys for classic gameplay and **touch controls**.
- Responsive design.
- Animated mobile menu.
- Score tracking.
- Best score memory. Stores highest score across sessions (**local storage**).
